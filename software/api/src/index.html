Welcome to PPE DATA INPUT API

Here's a table reviewing all available API methods and the path assigned to these methods:


CRUD Operations

Method Path Parameter Usage
POST /metrics none use this method on given path to add a new metric measurement to the Influx DB. In the post request send a JSON body containing following information: { "sensor": "height", "measurement": 3 }
GET /metrics none use this method on given path to retrieve all Influx DB entries from the last 60 minutes.
GET /stream none use this method on given path to obtain a continous stream of InfluxDB content.

Other Routes


Method Path Parameter Usage
GET /api-docs none obtain complete documentation of the API
GET /health/influx none obtain the health of the INFLUX DB instance
GET /health/api none obtain the health of the API

IMPORTANT: This application requires an InfluxDB to work perfectly. If you don't have one, you can use the one provided by using the Helm deployment in this repo.
