Here's a table reviewing all available API methods and the path assigned to these methods:


CRUD Operations

Method Path Parameter Usage
GET /api/stream none use this method on given path to connect to the continous data stream sending all data from the last hour stored in the InfluxDB

Other Routes


Method Path Parameter Usage
GET /api/health/influx none obtain the health of the INFLUX DB instance
GET /api/health/api none obtain the health of the backend

IMPORTANT: This application requires an InfluxDB to work perfectly. If you don't have one, you can use the one provided by using the Helm deployment in this repo.
